# Visual Constraints

Use this file for stable deck design rules.

## Typography

- Minimum body text: 18 pt unless the deck format requires otherwise.
- Minimum chart labels: 12 pt.
- Prefer message-led slide titles.

## Layout

- Keep margins consistent.
- Avoid cramped slides.
- Use one primary visual idea per slide.
- Avoid cards inside cards.

## Data Visualization

- Use chart types that match the comparison:
  - Time: line or timeline.
  - Category comparison: bar.
  - Composition: stacked bar or simple part-to-whole.
  - Flow: process or Sankey only when helpful.
- Label axes and units.
- Preserve source attribution for data.

## Accessibility

- Maintain readable contrast.
- Avoid tiny labels.
- Do not rely on color alone to communicate meaning.

## Imagery

- Images should clarify product, place, person, state, or emotion.
- Avoid generic stock-like imagery when the audience needs specificity.

## Brand Compliance

- Follow `_config/brand-guide.md`.
- Record project-specific exceptions in the active project.
