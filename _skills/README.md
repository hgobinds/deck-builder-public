# Deck Builder Skills

Use skills and tools by stage:

| Trigger | Tool/Skill | Why |
|---|---|---|
| Existing `.pptx` provided | `pptx` | Extract text, inspect thumbnails, preserve template structure |
| Build final presentation | `pptx` or `presentations` | Create or modify the actual deck |
| Export PDF | `pdf` | Create shareable/static version |
| Data-heavy slides | spreadsheet/data tools | Validate calculations and chart inputs |
| Current facts or external proof needed | web search/deep research | Gather sources, save relevant webpages, and ask user which sources to use |
| Custom visuals needed | image generation/design tools | Create icons, illustrations, textures, or backgrounds |
| Interactive deck or web slides | frontend/design tools | Build HTML or web-based presentation variant |
